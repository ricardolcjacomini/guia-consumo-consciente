## Como fazer o deploy no Vercel

### Opção 1 — Vercel CLI

```bash
npm install -g vercel
vercel login
vercel --prod
```

### Opção 2 — Vercel Dashboard

1. Acesse vercel.com e faça login
2. Clique em "Add New Project"
3. Arraste esta pasta inteira para o campo de upload
4. Clique em Deploy
